How This Works

The system ranks every stock and ETF in the universe against each other each week. It does not predict absolute prices. It asks: which candidates rank highest on momentum, quality, and trend right now? Weights shift based on the current market regime. After 30+ weeks the IC feedback loop nudges weights toward whichever factors are actually predicting returns.

Selection Factors

Factor Default Weight Signal
Momentum 50% (Risk-On) / 20% (Risk-Off) 12M-1M return rank (Jegadeesh-Titman). Skips last month to avoid short-term reversal.
Quality 25% (Risk-On) / 50% (Risk-Off) Sharpe ratio + low max-drawdown + low volatility — ranked cross-sectionally.
Trend 25% all regimes Above 200MA + golden cross + 4-quarter return consistency.
